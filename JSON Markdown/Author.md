```json
"Authors" : [
    "AuthorOne" : {
        "AuthorId" : "xx-xxx-xxx-xx",
        "Name" : "authorName", 
        "BookId" "xx-xxx-xxx-xx", 
    },
    "AuthorTwo" : {
        "AuthorId" : "xx-xxx-xxx-xx",
        "Name" : "authorName",

    "AuthorThree" : {
        "AuthorId" : "xx-xxx-xxx-xx",

```

### Access Single Author 
```https://Library/authors/{id}```
```json 
"Author" : {
        "AuthorId" : "xx-xxx-xxx-xx",
        "Name" : "authorName", 
        "BookId" "xx-xxx-xxx-xx", 
    },
```
